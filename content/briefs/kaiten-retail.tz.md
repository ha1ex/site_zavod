## Summary

Лендинг «Кайтен для розничных сетей» — Kaiten как российская система управления проектами, задачами и командами для розничной сети: магазины, закупки, ИТ, маркетинг и склады в одном пространстве. Режим custom (1-в-1 по ТЗ команды): порядок блоков и тексты строго из приложенного документа, лишние блоки не добавляются. Визуальный домен — pm (канбан-доски, карточки задач, сводный портфель проектов по магазинам, база знаний, отчёты), без e-commerce-артефактов вроде SKU, остатков и маркетплейсов — розница здесь про управление процессами сети, а не про товарный учёт. Логика по ТЗ: первый экран про единое пространство всей сети → бейджи доверия → прозрачность по магазинам и отделам → все отделы в одном сервисе → анатомия карточки задачи → крупные проекты на дочерних карточках → регламенты и документы из любого магазина → автоматические отчёты → четыре шага запуска за 30 минут → вопросы и ответы. Блока с отзывами/кейсами в ТЗ нет — не добавляем; METRO и сеть на 150+ точек упомянуты фактами в разделе вопросов и ответов.

## Аудитория

- **Основная:** Операционный или коммерческий директор розничной сети (10–200+ точек), который отвечает за прозрачность работы магазинов и отделов и хочет видеть статус по всей сети без ручного сбора отчётов
- **Принимает решение:** Операционный/коммерческий директор сети совместно с ИТ-директором (импортозамещение, установка на свои серверы, доступы для подрядчиков)
- **Роли:** Операционный / коммерческий директор розничной сети, Руководитель отдела закупок, ИТ-директор розничной сети, Руководитель магазина / куратор группы точек, Руководитель склада и логистики, Руководитель маркетинга сети

**Боли:**
- Руководитель видит статус по каждому магазину и отделу без переписок, звонков и ручных отчётов
- Задачи из Excel, Telegram и почты собираются на общих досках отделов
- Крупные проекты сети — маркировка, открытие точки, ребрендинг — разбиваются на дочерние карточки, прогресс по каждому шагу виден сразу
- Регламенты, техкарты и инструкции открываются с любого устройства прямо из магазина
- Отчёты по сети собираются автоматически, без сведения таблиц вручную

**Критерии выбора:**
- Российское ПО в реестре отечественного ПО и пригодность для импортозамещения
- Облако в России по 152-ФЗ и установка на собственные серверы
- Мобильное приложение для сотрудников магазинов без ноутбука
- Гибкие доступы: сотрудники отделов, кураторы магазинов, подрядчики и поставщики
- Автоматический перенос данных из Excel, Trello, Jira, Битрикс24 без ручного переноса

**Возражения:**
- У нас 100+ магазинов и разные отделы — справится ли система с таким масштабом
- Сколько времени займёт полное внедрение и можно ли переходить постепенно
- Есть ли мобильное приложение для сотрудников в торговом зале
- Не потеряем ли доступ, как с зарубежными сервисами
- Можно ли дать подрядчикам и поставщикам доступ только к их задачам
- Можно ли установить на свои серверы

### FAQ — вопросы и ответы
**Ответы на частые вопросы**

У нас 100+ магазинов и разные отделы — Кайтен справится с таким масштабом? Да. В Кайтен можно создать отдельное пространство для каждого отдела и магазина, а руководителю собрать сводный портфель проектов, где виден статус по всей сети.

Сколько времени занимает полное внедрение? От 1 до 1,5 месяцев в зависимости от масштаба — так было у METRO (закупки для 90+ магазинов) и у крупной торговой сети на 150+ точек. Переход можно делать постепенно, без остановки текущей работы: сначала один отдел, потом остальные.

Есть ли мобильное приложение для сотрудников в магазинах? Да. Приложения для iOS и Android с полным набором функций: обновление статусов, комментарии, просмотр документов и задач — без ноутбука.

Кайтен — российское ПО? Не потеряем доступ, как с зарубежными сервисами? Да, Кайтен внесён в реестр отечественного ПО (№14347), компания аккредитована как ИТ-организация. Облако размещено в России и соответствует 152-ФЗ, доступна и установка на свои серверы.

Можно ли пользоваться Кайтен бесплатно? Да. Бесплатный тариф не ограничен по сроку и числу пользователей, без лимитов на задачи и документы. При регистрации вы получаете 14 дней полного доступа ко всем функциям, без привязки карты. Все тарифы →

Как перенести данные из Excel, Trello, Jira или Битрикс24? Данные переносятся автоматически в один клик, структура сохраняется — не нужно создавать всё заново. Мы помогаем с настройкой пространств и переносом данных. Подробнее →

Можно ли дать подрядчикам и поставщикам доступ только к их задачам? Да. Создайте отдельное пространство для контрагента и ограничьте доступ так, чтобы он видел только свои задачи. Руководитель при этом видит всё в едином пространстве.

Можно установить на серверы? Да, есть полноценная коробочная версия. Данные хранятся на вашей инфраструктуре и не передаются третьим сторонам.
